NCATSFIND

Version 0.89

What is it?


NCATSFind is an opensource browser addon that helps you resolve names and images to structures with chemical sense. It uses the NCATS resolver webservice, which pulls from a variety of public databases.

How do I use it?

  • Manual Name Resolution
  • Automatic ID Resolution
  • Structure Image Recognition

Manual Name Resolution

You can resolve a single string of text using NCATSFind.
  • Select text to resolve with cursor.
  • From the popup, you can see the structure, and the source for the resolution.
  • Clicking the structure image will open a new tab, where you can edit / export / copy the structure.

Automatic ID Resolution

Beyond manually selecting names to resolve, you can also automatically highlight certain substance codes and identifiers.
  • Hovering-over these identifiers attempts to load a structure image to display.
  • Clicking on the icon will load the resolver popup.
At present, NCATSFind can identify these formats in a document
  • UNII
  • CAS Registry Number
  • NCGC IDs
  • InChI [testing]
These can be customized in NCATSFind settings, which can be launched from the toolbar.

Structure Image Recognition

NCATSFind also uses a server-side service based on OSRA to make chemical sense of structural images.
  • Click the Toolbar icon
  • Click the "Snapshot to Structure" button
  • Click and drag to select the area of interest.
  • The resolution will open in an editor tab

What libraries does it use?

At present, the addon uses the following client-side javascript libraries (and their dependencies):
  • JQuery
  • JQueryUI
  • Zepto
  • Ketcher
  • JSGrowl
